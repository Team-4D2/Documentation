Welcome to UniFeed's documentation!
===================================

This is the technical documentation for UniFeed, a social media platform built using FastAPI. This documentation provides an overview of the project, its architecture, and how to use it.
The Fountend is built using Flutter, a popular framework for building cross-platform mobile applications. The backend is built using FastAPI, a modern web framework for building APIs with Python. The database used is SQLite, which is a lightweight and efficient database engine.


.. note::

   This project is under active development.
